Process in memory 方向

系统结构

  • 神经元接口（前端电路）→ ADC → PIM → 控制逻辑（控制刺激或者通信）→通信电路（刺激电路）
  • 供能模块

约束条件

  • 面积：芯片面积越大，接收到的能量越多，同时可以容纳更多的RAM单元，也可以拥有更强的算力，初步想法是芯片的面积约$10mm^2$，至少能实现多通道神经元信号分离、简单模式识别。
  • 能量：能量约束取决于能量供给模块，同时应该采用高能效比的体系架构，在植入性设备上，能效比 > 性能

确定怎样的应用可以使用PIM

  • 神经信号的模式识别
  • 神经信号处理算法

有待进一步调研的问题

  • 在植入式神经芯片的应用中，最大的能量消耗是哪一个部分，根据经验ADC会消耗大量能量，而模式识别或者数据处理算法所消耗的能量占多大比例（至少要超过30%才有使用PIM的意义）
  • 与传统架构相比，使用PIM所额外增加的控制电路面积与功耗（植入性应用中RAM单元较少，所以相应控制电路占比会更大）

  • PIM在植入式芯片上是否有优势，一般来说PIM会在 memory-intensive的应用上具有优势，比如当数据在内存和处理器之间大量搬运时（NN运算），在功耗和电路面积极其受限的情况下，PIM是否有用

  • 神经信号的处理算法、神经信号模式识别算法、神经疾病诊断

和展宏聊天结果

  • PIM不是目的，而是降低功耗、提高能效比的手段之一
  • 植入式计算芯片的目的是在与可以直接本地计算，减少与体外系统的通信
    • 计算电路与通信电路功耗比较
    • 算法适合何种器件、电路、体系结构
